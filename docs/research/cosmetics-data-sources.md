## 1. 국내 데이터 소스

### 1.1 화해 (Hwahae)

| 항목 | 내용 |
|------|------|
| **URL** | https://www.hwahae.co.kr |
| **데이터 범위** | 국내 유통 화장품 약 20만+ 제품 |
| **전성분 제공** | O (한국명 + INCI명 병기) |
| **성분 분석** | EWG 등급, 자체 성분 등급 |
| **리뷰 데이터** | 피부 타입별 리뷰, 평점 |
| **API 제공** | X (공개 API 없음) |

**수집 가능성:** 낮음
**법적 이슈:** 화해의 성분 데이터는 자체 DB로 구축한 것. 웹 스크래핑은 서비스 이용약관 위반 및 저작권 침해 소지. 화해와 데이터 파트너십 협의가 필요.
**데이터 품질:** 매우 높음 (전문 팀이 성분 검증)
**COSFIT 활용:** 파트너십 체결 시 성분 매핑 검증 데이터로 활용 가능

## 2. 글로벌 데이터 소스

| **데이터 범위** | 글로벌 화장품 성분 DB |
| **전성분 제공** | O (INCI명 기준) |
| **성분 분석** | 성분별 기능, 자극도, 안전성 상세 설명 |

**수집 가능성:** 낮음 (스크래핑 금지)
**법적 이슈:** 저작권 보호 콘텐츠
**데이터 품질:** 매우 높음 (화학자가 직접 작성한 성분 설명)
**COSFIT 활용:** 성분 지식 DB 구축 시 참고 자료. Ingredient.knowledgeData 보강에 수동 참조 가능.

### 2.2 CosDNA

| **전성분 제공** | O (사용자 등록 기반) |

**수집 가능성:** 낮음

**COSFIT 활용:** Ingredient 모델의 cosDnaIrritant, cosDnaSafety 필드에 매핑 (수동 또는 라이선스 협의)

### 2.3 EWG Skin Deep

| 항목 | 내용 |
|------|------|
| **URL** | https://www.ewg.org/skindeep |
| **데이터 범위** | 약 90,000+ 제품, 2,500+ 성분 |
| **전성분 제공** | O |
| **성분 분석** | EWG Hazard Score (1~10) |
| **API 제공** | X (데이터 라이선스 별도 문의) |

**수집 가능성:** 낮음 (라이선스 구매 필요)

**COSFIT 활용:** Ingredient.ewgScore 필드. EWG 공식 등급표를 참조하여 성분별 등급 수동 입력 가능.

### 2.4 Open Beauty Facts

| 항목 | 내용 |
|------|------|
| **URL** | https://world.openbeautyfacts.org |
| **데이터 범위** | 오픈소스 화장품 DB (약 10,000+ 제품) |
| **전성분 제공** | O (사용자 기여) |
| **성분 분석** | 기본 분석 |
| **API 제공** | **O (무료, 오픈소스)** |
| **라이선스** | Open Database License (ODbL) |

**수집 가능성:** 높음
**법적 이슈:** 오픈 라이선스, 출처 표기 및 동일 조건 공유 필요
**데이터 품질:** 낮음~중간 (사용자 기여라 불완전, 한국 제품 적음)
**COSFIT 활용:**
- API 엔드포인트: `https://world.openbeautyfacts.org/api/v2/product/{barcode}.json`
- 바코드 기반 제품 검색 가능
- 글로벌 제품 보완 데이터로 활용
- 한국 제품은 거의 없어 주력 소스로는 부적합

## 4. 권장 데이터 수집 전략

### Phase 1: 식약처 API 최적화 (현재)
- I0030 전체/일일 동기화 안정화 (현재 구현 완료)
- 성분 정규화 파이프라인 강화 (IngredientSynonym 활용)
- 매칭률 모니터링 대시보드

### Phase 2: 관리자 수동 입력 시스템 (이번 작업)
- 관리자 제품 추가 페이지 구현
- 전성분 텍스트 파싱 + DB 매칭
- 미매칭 성분 수동 매핑 UI

### Phase 3: 보조 API 연동
- **Open Beauty Facts** API 연동 (바코드 기반 글로벌 제품 보완)
- **PubChem** API 연동 (CAS Number 기반 성분 지식 자동 보강)
- 식약처 I0040 (기능성화장품) 추가 연동
- 식약처 원료 기준 DB → safetyGrade 자동 산정

### Phase 4: 데이터 파트너십
- 화해/글로우픽 데이터 파트너십 협의
- 올리브영 파트너 API 연동 (입점사 제품 자동 동기화)
- 브랜드 공식 사이트 제품 정보 API 협의

## 5. 법적 고려사항 요약

1. **스크래핑 금지:** 화해, 글로우픽, 올리브영 등 상용 서비스의 웹 스크래핑은 이용약관 위반 및 저작권법/부정경쟁방지법 위반 소지
2. **공공 데이터 활용:** 식약처 API, Open Beauty Facts, PubChem은 법적으로 안전하게 활용 가능
3. **EWG 등급 참조:** EWG 점수를 직접 표시하려면 라이선스 협의 필요. 자체 등급 체계 구축 권장
4. **사용자 기여 데이터:** 관리자가 직접 입력하는 제품/성분 정보는 법적 이슈 없음
5. **API 이용약관:** 각 API 제공기관의 이용약관 준수 필수 (출처 표기, 상업적 이용 조건 등)
